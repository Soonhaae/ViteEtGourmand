# Documentation technique

Ce projet est une application web dynamique pour permettre la gestion et l’affichage de contenus dynamique, avec un système d’interactions, côté client et côté serveur.

## Technologies utilisées

### front-end

- HTML5
- CSS3
- Bootstrap
- javascript

### back-end

- PHP
- MySQL (via JawsDB)
- Apache 2
- MongoDB

## Installation du projet

### cloner le dépôt

git clone https://github.com/Soonhaae/ViteEtGourmand

### Heroku

installer le buildpack heroku/php
installer le add-on JawsDB MySQL
choisir en "Deployment method" : GitHub Projet Vite&Gourmand
